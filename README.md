HR Analytics Dashboard

This project presents an interactive HR Analytics Dashboard designed to provide insights into workforce demographics, employee distribution, and salary trends. The dashboard enables data-driven decision-making by visualizing key HR metrics in a clear and intuitive format.

🚀 Key Features
High-Level KPIs
Total Employees: 2M
Total Age Sum: 55K
Employee Count: 1480
Education Score Sum: 4308

Demographic Analysis
Gender-based comparison of employee age distribution
Age group segmentation (18–25, 26–35, 36–45, etc.)
Education distribution across different age groups

Employee Insights
Employee count breakdown by Employee ID
Filtering capabilities by:
Employee Number
Education level
Age range

Compensation Trends
Visualization of Percent Salary Hike by Job Role
Comparative analysis across roles such as:
Sales Executive
Research Scientist
Manager
Human Resources

📈 Visualizations Included
Bar charts for gender and age comparisons
Pie chart for education distribution by age group
Donut chart for employee distribution
Line chart for salary hike trends by job role
Interactive slicers for dynamic filtering

🛠️ Purpose
The dashboard is built to:
Analyze workforce composition
Identify trends in employee demographics
Support HR decision-making with data insights
Monitor compensation patterns across roles

📌 Use Cases
HR reporting and analytics
Workforce planning
Salary and compensation analysis
Diversity and inclusion tracking
